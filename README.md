# Project Assistant

A clean and organized repository for hosting custom scripts, website customizations, and styling experiments.

## 🚀 Features
* Custom stylesheet modifications for learning platforms.
* Clean, human-readable JavaScript code files.
* Organized structure for safe testing and backup storage.

## 🛠️ How to Use
1. Open the target webpage in your browser.
2. Open your browser console or extension manager.
3. Load the corresponding script file from this repository to test your custom elements.
